# Capstone Project

Capstone project repository for the **Frontend AI Engineering** internship track.

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js

## Getting Started

Once the React app scaffolding is in place:

```bash
cd client
npm install
npm start
```

The app will be available at `http://localhost:3000` by default.

## Development

This project is built with AI-assisted coding tools (e.g., Cursor) as part of the internship workflow. Commits follow the [Conventional Commits](https://www.conventionalcommits.org/) format — see [CLAUDE.md](./CLAUDE.md) for full coding conventions.
